# ⚙️ AIS-OS — Seu AI Operating System no Claude Code

Curso gratuito (formato INEMA.CLUB) que transforma o **Claude Code** num **AI Operating System (AIOS)** pessoal: um sistema que conhece seu negócio, alcança suas ferramentas, sabe fazer o trabalho e roda sem ser pedido.

**🌐 Acesse o curso:** https://inematds.github.io/ais-os/

Baseado no kit open-source **AIS-OS** (licença MIT) de Nate Herk — destrinchado em trilhas, módulos e tópicos com diagramas SVG, modo claro/escuro e navegação completa.

---

## O que você aprende

### 🧠 Trilha 1 — O Cérebro do Operador (3 Ms) · *como pensar, decidir e construir*
- **1.1 Mindset** — Default Shift, Function Breakdown, Curiosity Rule, o "Dip", fail fast.
- **1.2 Method** — Find the Constraint, EAD, regra 60/30/10, Map the Process, Autonomy Spectrum L0→L4, KPI.
- **1.3 Machine** — Lego Principle, Assembly Line, Validation Chain, Bike Method, Intern Rule, Kill Switch.

### 🏛️ Trilha 2 — A Arquitetura (4 Cs) · *o que você constrói*
- **2.1 Context** — a fundação não-pulável: `context/`, `CLAUDE.md`, `voice.md`, `decisions/`.
- **2.2 Connections** — os 7 domínios Tier-1, mecanismos (MCP/script/export/key+ref), `connections.md`.
- **2.3 Capabilities** — skills vs agents, anatomia de um `SKILL.md`, determinístico vs IA.
- **2.4 Cadence** — triggers recorrentes, rituais, "Cadence é por último", os 3 indicadores de sucesso.

### 🛠️ Trilha 3 — Operar na Prática · *as 3 skills + como crescer*
- **3.1 `/onboard`** — setup do Dia 1, as 7 perguntas, a regra da voz, o "wow moment".
- **3.2 `/audit`** — diagnóstico Four-Cs, pontuação 0–100, stages, lacunas por alavancagem.
- **3.3 `/level-up`** — 1 automação por semana, as 3 fases, Bike Method lock, brain-rewire.
- **3.4 Expansões & Cadência** — o que adicionar, anti-padrões, a regra dos 3 sins.

**3 trilhas · 11 módulos · ~70 tópicos · ~7 horas.**

---

## Como navegar

- Abra `index.html` (landing) ou vá direto a uma trilha em `curso/trilhaN/index.html`.
- Cada índice de trilha tem o **Mapa da trilha** + cards de módulo com tópicos expansíveis e modais.
- Cada módulo é uma página completa com diagrama SVG, exemplos e resumo.

Site estático, sem build. Funciona abrindo os arquivos no navegador ou via GitHub Pages.
